Przykladowe mozliwosci Markdown.

Naglowek tworzymy znaczkiem "#". Im wiecej "#" tym wiekszego rzedu jest naglowek. Przyklad:

Zwykly tekst.
# Tekst naglowka.
### Tekst naglowka trzeciego stopnia.

----------------------------------------

Kursywe uzyskujemy dzieki "*" lub "_", a pogrubienie dzieki "**" lub "__" umieszczonym przed i za zdaniem. Przyklad:

Zwykly tekst
*Tekst pochylony*
**Tekst pogrubiony**
*Pomieszane __rodzaje__ tekstu*

----------------------------------------

W Markdown mozemy tworzyc listy uporzadkowane i nieuprzadkowane. Nieuprzadkowane korzystaja z "*" a uprzadkowane z numerow i kropek "1." oraz "*". Przyklad:

* banan
* jablko
  * pomarancza
